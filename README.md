# AlchAss 功能介绍

## 基本概念

- 路径：加入材料后，游戏中显示的搅拌移动路径虚线
- 加水：倒入基底时，药瓶移动的方向直线
- 骷髅区：地图上会扣血的区域，也包括酒基地图上回血的治疗区
- 沼泽区：油基地图上会使药瓶移动时打滑的区域
- 碰撞偏移：游戏中存在的漏洞，当进行某些操作后，会出现药瓶的坐标位置和碰撞体位置不同的情况。游戏中大多数判定（如漩涡、骷髅区、沼泽区等）都使用碰撞体位置，但效果等级判定使用坐标位置
- 效果等级：L0 对应无法获取效果，L1 至 L3 分别对应可获取 I 至 III 级效果。按 III 级效果最大偏差程度为 100% 折算总体偏差比例 P，P <= 100% 时可获得 III 级；100% < P <= 600% 时可获得 II 级；P > 600% 且位置偏差比例 P_p <= 2754% 时可获得 I 级

## 按键操作

- 可在 `config` 文件中自定义快捷键
- F2：开关功能面板
- CTRL + 右键单击：将点击到的效果选为目标效果；若未点击到任何效果，取消目标效果选择
- ALT + 右键单击：将点击到的漩涡选为目标漩涡；若未点击到任何漩涡，取消目标漩涡选择
- SHIFT：高亮光标位置附近的自定义方向线。需要激活自定义方向线
- SHIFT + 右键单击：删除高亮的自定义方向线；若无高亮方向线，则新建一条通过当前鼠标位置和药瓶位置的方向线。需要激活自定义方向线
- SHIFT + 中键长按：拖拽以旋转高亮的自定义方向线。需要激活自定义方向线

## 功能面板

### 渲染选项

- 路径切向线：渲染路径方向所在的直线，即路径曲线的切线。需要激活路径曲线
- 加水方向线：渲染加水方向所在的直线，即药瓶与原点的连线。若存在目标效果、目标漩涡，会显示加水路径与它们的最近点
- 效果径向线：渲染药瓶与目标效果的连线
- 漩涡径向线：渲染药瓶与目标漩涡的连线
- 漩涡切向线：渲染药瓶沿漩涡运动的方向直线，即漩涡曲线的切线。需要激活漩涡曲线
- 自定义方向线：渲染通过药瓶位置的，自定义方向的直线
- 路径曲线：用两种颜色交替渲染路径曲线。对于矿石路径，使用一条直虚线段表示。若存在目标效果、目标漩涡，会显示路径与它们的最近点。需要存在路径
- 漩涡曲线：渲染药瓶在当前漩涡中加热时的运动轨迹。需要在漩涡中
- 效果范围：渲染目标效果的 I 至 III 级效果范围圈，药瓶中心须在圈内才能获取相应等级的效果。会随药瓶与效果间旋转角度的差异改变圈的大小；当开启路径曲线、加水方向线时，会显示范围圈与它们的交点。需要存在目标效果
- 漩涡范围：渲染目标漩涡的范围圈，药瓶中心须在圈内才能被漩涡影响。当开启路径曲线、加水方向线时，会显示范围圈与它们的交点。需要存在目标漩涡
- 骷髅区模拟：渲染路径、加水、漩涡路径上进出骷髅区的位置，和可能存在的死亡点的位置。分别需要开启路径曲线、加水方向先、漩涡曲线
- 沼泽区模拟：渲染路径上进出沼泽区的位置，并将路径按实际移动轨迹缩放。需要在油基地图，且开启路径曲线
- 透明瓶身：使用一个圆圈代替药瓶渲染。圆圈的大小与药瓶的判定箱一致，使用一条半径指示药瓶朝向

### 显示选项

- 盐量数据：切换信息窗口中旋转角度的显示方式（原始角度/折算盐量）
- 搅拌阶段：切换信息窗口中搅拌进度的显示方式（阶段与进度之和/分别显示阶段和进度）
- 极坐标：切换信息窗口中坐标的显示方式（直角坐标/极坐标）
- 碰撞体位置：切换模组对于药瓶位置的计算方式（原始坐标/碰撞偏移下实际坐标）

### 窗口管理

- 恢复所有窗口：将功能面板和所有信息窗口打开，并恢复到默认位置
- 保存窗口布局：将功能面板和信息窗口当前的位置保存为默认位置

### 自定义方向线管理

- 添加方向线：添加一条默认的自定义方向线
- 方向线条目：浏览和修改线的方向、删除方向线

## 信息窗口

### 路径信息

- 需要激活路径曲线
- 总体偏离：路径上距离目标效果最近点，与效果位置间的距离，和药瓶与效果旋转角度差值的加权和，折算为效果等级和偏差百分比。需要存在目标效果
- 位置偏离：路径上距离目标效果最近点与效果位置间的距离，折算为效果等级和偏差百分比。需要存在目标效果
- 近点方向：路径上距离目标效果最近点，相对于目标效果位置的方向。需要存在目标效果
- 效果夹角：路径切向线与效果径向线的夹角。需要激活路径切向线、效果径向线
- 加血需求：沿路径下次离开骷髅区前还需加血的量，负值表示离开前剩余血量。需要激活骷髅区模拟
- 沼泽长度：沿路径下次离开沼泽区前，还需在沼泽中移动的距离。需要激活路径曲线、沼泽区模拟

### 加水信息

- 需要激活加水方向线
- 总体偏离：加水路径上距离目标效果最近点，与效果位置间的距离，和药瓶与效果旋转角度差值的加权和，折算为效果等级和偏差百分比。需要存在目标效果
- 位置偏离：加水路径上距离目标效果最近点，与效果位置间的距离，折算为效果等级和偏差百分比。需要存在目标效果
- 近点方向：加水路径上距离目标效果最近点，相对于目标效果位置的方向。需要存在目标效果
- 效果夹角：加水方向线与效果径向线的夹角。需要激活效果径向线
- 加血需求：沿加水路径下次离开骷髅区前还需加血的量，负值表示离开前剩余血量。需要激活骷髅区模拟

### 效果偏离

- 需要存在目标效果
- 总体偏离：药瓶与效果位置间的距离，和药瓶与效果旋转角度差值的加权和，折算为效果等级和偏差百分比
- 位置偏离：药瓶与效果位置间的距离，折算为效果等级和偏差百分比
- 旋转偏离：药瓶与效果旋转角度差值，折算为效果等级和偏差百分比

### 目标效果

- 需要存在目标效果
- 目标效果：目标效果名称
- 坐标位置：目标效果位置
- 旋转角度：目标效果旋转角度
- 效果方向：目标效果相对于药瓶的方向。需要激活效果径向线

### 移动信息

- 搅拌进度：当前已经搅拌的距离
- 路径方向：当前路径的方向。需要激活路径切向线
- 加水方向：当前加水的方向。需要激活加水方向线
- 漩涡切角：当前所在漩涡中，加水贴边的最大/小可能角度。以加水方向计。需要在漩涡中

### 酿造信息

- 坐标位置：药瓶的位置
- 碰撞偏移：药瓶碰撞体位置相对于坐标位置的坐标
- 旋转角度：药瓶旋转的角度
- 当前血量：药瓶的血量

### 研磨信息

- 研磨进度：研钵中药材的研磨程度。需要研钵中存在药材

### 活跃漩涡

- 需要在漩涡中
- 漩涡距离：药瓶到当前漩涡中心的距离
- 最大距离：当前漩涡的半径
- 漩涡方向：当前漩涡相对于药瓶的方向
- 漩涡切向：沿当前漩涡加热移动的路径方向。需要激活漩涡切向线
- 加血需求：沿当前漩涡路径下次离开骷髅区前还需加血的量，负值表示离开前剩余血量。需要激活漩涡切向线、骷髅区模拟

### 目标漩涡

- 需要存在目标漩涡
- 漩涡距离：药瓶到目标漩涡中心的距离
- 路径近点：路径上离目标漩涡最近点的距离。需要激活路径曲线
- 加水近点：加水路径上离目标漩涡最近点的距离。需要激活加水方向线
- 最大距离：目标漩涡的半径
- 漩涡方向：目标漩涡相对于药瓶的方向。需要激活漩涡径向线
